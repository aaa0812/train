# Train

## Compilation
C’est le fichier TD_train_main qui doit être build.
L’exécutable se trouvera dans le dossier bin. Il est important qu’il soit exécuté de là, pour que les fichiers nécessaires soient trouvés.

## Commande

| Commande                 | Résultat                                         |
| :------------------------|:-------------------------------------------------|
| Z, Q, S, D               | avancer, aller à gauche, reculer, aller à droite |
| Flèches directionnelles  | Orientation de la caméra                         |
| L                        | Visualisation du mesh                            |
| P                        | Mode rempli                                      |
| R                        | Apparition de la grille au sol                   |
| Échap                    | Fermeture de la fenêtre                          |



## Fonctionnalités implémentées 
Affichage de la grille, Rails droits et courbés positionnés en fonction du fichier json, éléments de décor (levier, lanternes, potence en bois…), gare, caméra FPS, lumières (scène, à l’avant du train, en fonction des lanternes).
